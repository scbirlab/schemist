# Chemical string format converter

Trivial batch interconversion of 1D chemical formats.

Frontend for [schemist](https://github.com/scbirlab/schemist) to allow interconversion from:

- SMILES
- SELFIES
- Amino acid sequences
- HELM

to...

- Strucure image
- SMILES
- SELFIES
- InChI
- InChIKey
- Name 
- cLogP
- TPSA
- molecular weight
- charge

... and several others!
